Lesson 1 Notes

What is the web, and what makes up the web?

The web is a huge network of computers, that all share information with each other. Anything can exist on the web, including videos, images, HTML, and Hyper-links.Hyper-Links are links between web pages, such as this one that links to Google.com.
There are four major parts of the web, which are computers, the Internet, servers, and the Internet browser. Computers are machines which run your web browser, that connects to and displays files from the Internet. Which is a huge network of servers also known as huge computers that hold gigabytes of data, and is always on and running.
HTTP is the protocol that is used to link and communicate with servers.

What is HTML?

HTML stands for Hypertext Mark-up Language . HTML is the code that most if not every page on the Internet is written in. HTML is made up of text content, which is what you see on a web page, mark up, which is what defines how your content is laid out, and references or links to other documents.
HTML Mark up, is made up of elements, and tags

What are Elements and Tags

Elements are the format in which HTML code is written. An element contains the following an opening tag, the contents of the tag, and the closing tag.
Elements also come in two different formsIn-line elements, and Block elements.The difference between the two, is that block elements put a invisible box around the text, whereas in-line elements do not do this.
Tags are the commands used to format a page, they are usually contained within angled brackets. There are 2 types of tags the first type is a tag that has an opening and closing tag, and there is another type of tag called a void tag. They are tags that only have an opening tag such as the break tag. whereas the paragraph tag would be a normal tag.

Browser developer tools

Every browser that you encounter will most likely contain some sort of way for developers to see the code is behind the web page that is being displayed. If you use chrome like I do then all you have to do is hit your F12 key on your keyboard. This will bring up the Developer Console, which will allow you to see the code that is behind the page your on, it will also show you this in a tree like structure with arrows that show and hide different parts of the page. I will talk about this more in another part of this lesson's notes

What is CSS?

CSS stands for Cascading Style Sheet, and it is a styling language. It is used for defining the look and feel of an HTML page, and it does this by using syntax tools.

What is a DOM?

DOM stands for Document Object Model. It is basically the tree structure which I was talking about earlier. It represents they way your HTML code is organized, and how your code is represented in the developer console. If your code is organized then your code should be very easy to read and understand. Examples of this may be the div tag which puts a box around each part of your code that you want to keep organized. Each of these boxes are labeled with a class tag within each div element.
